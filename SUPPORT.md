# Support

## Supported Scope

Support currently focuses on:

- the active default branch
- the latest Windows desktop production release
- the documented backend CLI entry points
- the shared archive core used by desktop and CLI

Older snapshots may not receive fixes unless the issue is security-critical and practical to backport.

## Getting Help

Use GitHub issues for:

- installation problems
- reproducible bugs
- documentation gaps
- feature requests

Feature requests are treated as proposals, not roadmap commitments. Avikal is maintainer-led, and final prioritization remains with the maintainer.

Use the private process in [SECURITY.md](./SECURITY.md) for suspected vulnerabilities.

## Good Reports Include

- Avikal version or commit
- operating system and architecture
- whether the issue affects desktop, CLI, TimeCapsule, Aavrit, or packaging
- exact reproduction steps
- expected behavior
- actual behavior
- relevant logs or screenshots

Do not include passwords, keyphrases, `.avkkey` files, private archives, tokens, or decrypted data in public issues.
